=============================================================================
  🎨 Apple 风格设计规范
=============================================================================

字体系统:
  - macOS/iOS: SF Pro Display/Text
  - 降级方案: Inter → Segoe UI → Roboto
  - 字体平滑: -webkit-font-smoothing: antialiased
  - 字体渲染: text-rendering: optimizeLegibility

颜色系统:
  浅色主题:
    - 背景: 白色 hsl(0 0% 100%)
    - 文本: 深灰 hsl(0 0% 13%)
    - 卡片: 白色 hsl(0 0% 100%)
    - 激活状态: 浅蓝 hsl(211 98% 95%)
    - 系统蓝色: hsl(211 98% 52%)

  深色主题:
    - 背景: 深黑 hsl(0 0% 7.5%)
    - 文本: 白色 hsl(0 0% 100%)
    - 卡片: 深灰 hsl(200 5% 11.5%)
    - 激活状态: 深灰 hsl(210 10% 25%)
    - 系统蓝色: hsl(211 98% 52%)

对比度标准:
  - 普通文本: ≥ 4.5:1 (WCAG AA)
  - 大文本: ≥ 3:1 (WCAG AA)
  - UI 组件: ≥ 3:1 (WCAG AA)

  🔍 技术实现细节
=============================================================================

修复方案: 全局 CSS 覆盖策略

优先级顺序:
  1. global-color-fix.css (最高优先级，使用 !important)
  2. button-fix.css
  3. accessibility.css
  4. fonts.css
  5. index.css
  6. Tailwind 默认样式
  7. 组件内联样式

主题变量映射:
  bg-slate-50/100    → --card (卡片背景)
  bg-slate-200/300   → --muted (次要背景)
  bg-slate-600/700   → --secondary (次要背景)
  bg-slate-800/900   → --card (卡片背景)
  text-white         → --primary-foreground
  text-slate-400/500 → --muted-foreground (次要文本)
  text-slate-600/700 → --foreground (主要文本)
  border-slate-*     → --border (边框)
