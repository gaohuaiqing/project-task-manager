# 测试用例 - 报表查看与导出

> **模块**: Reports
> **优先级**: P1
> **场景数**: 3

---

## TC-RPT-01：查看仪表板统计

### 场景描述

验证仪表板显示的统计数据正确

### 前置条件

- 已登录 admin 或 dept_manager 或 tech_manager
- 系统中存在项目和任务数据

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入仪表板页面 | 页面加载完成 |
| 2 | 检查统计卡片 | 显示任务总数、完成数、延期数等 |
| 3 | 验证数据准确性 | 统计数字与实际数据一致 |
| 4 | 检查趋势图 | 显示最近 7 天的趋势数据 |
| 5 | 检查预警列表 | 显示延期预警和已延迟的任务 |

---

## TC-RPT-02：查看各类报表

### 场景描述

验证 5 种报表能正常加载和显示数据

### 前置条件

- 已登录有报表权限的用户（admin 或 50223183 或 50234447）

### 报表类型

1. 项目进度报表
2. 任务统计报表
3. 延期分析报表
4. 成员任务分析
5. 资源效能分析

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入报表页面 | 显示报表标签页 |
| 2 | 依次点击各报表标签 | 每个报表正常加载 |
| 3 | 检查报表数据 | 显示正确的统计数据和图表 |
| 4 | 使用筛选条件 | 筛选后数据正确更新 |

---

## TC-RPT-03：导出报表数据

### 场景描述

验证报表数据可以导出为 Excel

### 前置条件

- 已登录有报表权限的用户
- 报表页面已加载

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 在报表页面点击"导出"按钮 | 弹出导出选项 |
| 2 | 选择导出格式（Excel） | 开始下载 |
| 3 | 打开下载的 Excel 文件 | 文件正常打开 |
| 4 | 检查 Excel 内容 | 数据与页面显示一致 |
