## TC-PROJ-01：创建项目

### 场景描述

创建新项目并验证必填字段校验

### 前置条件

- 已登录有项目创建权限的用户（admin 或 50223183 或 50234447）

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入项目管理页面 | 显示项目列表 |
| 2 | 点击"新建项目"按钮 | 弹出新建项目对话框 |
| 3 | 填写项目名称 | 输入成功 |
| 4 | 选择项目类型 | 选择成功 |
| 5 | 选择开始日期和结束日期 | 选择成功 |
| 6 | 点击保存 | 提示创建成功 |
| 7 | 检查项目列表 | 新项目显示在列表中 |

### 必填字段校验

| 场景 | 预期结果 |
|------|---------|
| 项目名称为空 | 提示"请输入项目名称" |
| 项目类型未选择 | 提示"请选择项目类型" |
| 开始日期未选择 | 提示"请选择开始日期" |
| 结束日期早于开始日期 | 提示"结束日期不能早于开始日期" |

## TC-PROJ-02：项目成员管理

### 场景描述

添加和移除项目成员

### 前置条件

- 已登录有项目编辑权限的用户
- 项目已创建

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入项目详情页 | 显示项目信息 |
| 2 | 点击"成员管理"标签 | 显示成员列表 |
| 3 | 点击"添加成员"按钮 | 弹出成员选择器 |
| 4 | 选择成员并确认 | 成员添加成功 |
| 5 | 检查成员列表 | 新成员显示在列表中 |
| 6 | 点击成员的"移除"按钮 | 弹出确认对话框 |
| 7 | 确认移除 | 成员移除成功 |
