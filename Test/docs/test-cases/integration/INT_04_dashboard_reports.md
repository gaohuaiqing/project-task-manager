# 集成测试：仪表板→报表联动

> **场景**: 仪表板与报表数据联动验证
> **用例数**: 5
> **层级**: L2 集成测试

---

## INT-DASH-RPT-01: 仪表板数据与报表一致性

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证仪表板统计数据与报表数据一致 |
| **前置条件** | 已登录为管理员，系统有数据 |
| **操作步骤** | 1. 查看仪表板统计卡片 → 2. 进入任务统计报表 → 3. 对比数据 |
| **预期结果** | 仪表板任务数与报表任务总数一致 |
| **验证点** | 数值完全匹配 |

---

## INT-DASH-RPT-02: 紧急任务跳转到任务详情

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证仪表板紧急任务可跳转到任务详情 |
| **前置条件** | 存在延期预警任务 |
| **操作步骤** | 1. 查看仪表板紧急任务提醒 → 2. 点击任务项 |
| **预期结果** | 跳转到任务管理页面，定位到该任务 |
| **验证点** | 任务高亮显示或滚动到可视区域 |

## INT-DASH-RPT-04: 权限范围过滤一致性

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证仪表板和报表对相同权限用户显示相同数据范围 |
| **前置条件** | 已登录为部门经理，存在跨部门数据 |
| **操作步骤** | 1. 查看仪表板项目数 → 2. 进入项目进度报表 → 3. 对比项目列表 |
| **预期结果** | 仪表板和报表显示相同的项目列表 |
| **验证点** | 数据范围与用户权限完全一致 |

---

## INT-DASH-RPT-05: 实时数据更新验证

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证数据变更后仪表板和报表同步更新 |
| **前置条件** | 已登录为管理员 |
| **操作步骤** | 1. 记录仪表板任务数 → 2. 创建新任务 → 3. 刷新仪表板 → 4. 进入报表对比 |
| **预期结果** | 仪表板和报表的任务数都增加1 |
| **验证点** | 两处数据同步更新，无延迟 |
