# 测试用例 - 任务导入导出

> **模块**: Import/Export
> **优先级**: P2
> **场景数**: 2

---

## TC-IMP-01：导入任务数据

### 场景描述

批量导入任务数据到系统中

### 前置条件

- 已登录有导入权限的用户
- 准备好符合模板格式的 Excel 文件

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入任务管理页面 | 显示任务列表 |
| 2 | 点击"导入"按钮 | 弹出导入对话框 |
| 3 | 下载导入模板 | 模板下载成功 |
| 4 | 填写任务数据到模板 | 数据填写完成 |
| 5 | 选择填写好的文件并上传 | 文件上传成功 |
| 6 | 检查导入预览 | 显示待导入的任务列表 |
| 7 | 点击确认导入 | 提示导入成功，显示成功数量 |
| 8 | 检查任务列表 | 新导入的任务显示在列表中 |

### 异常场景

| 场景 | 预期结果 |
|------|---------|
| 文件格式不正确 | 提示"文件格式错误" |
| 必填字段为空 | 提示具体字段错误 |
| 项目不存在 | 提示"项目不存在" |

## TC-IMP-02：导出任务数据

### 场景描述

将任务数据导出为 Excel 文件

### 前置条件

- 已登录有导出权限的用户
- 系统中存在任务数据

### 操作步骤与验证点

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 进入任务管理页面 | 显示任务列表 |
| 2 | 设置筛选条件（可选） | 筛选条件生效 |
| 3 | 点击"导出"按钮 | 弹出导出选项 |
| 4 | 选择导出格式（Excel） | 开始下载 |
| 5 | 打开下载的 Excel 文件 | 文件正常打开 |
| 6 | 检查 Excel 内容 | 数据与页面显示一致 |
