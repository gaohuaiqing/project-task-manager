| 项目 | 内容 |
|------|------|
| **测试目的** | 验证管理员从登录到查看报表的完整流程 |
| **前置条件** | 系统运行正常，测试数据库已初始化 |
| **操作步骤** | 1. 登录admin账号<br>2. 创建项目E2E-FULL-001<br>3. 添加成员engineer1到项目<br>4. 创建里程碑<br>5. 创建根任务和子任务<br>6. 分配任务给engineer1<br>7. 查看仪表板确认数据<br>8. 进入报表查看统计 |
