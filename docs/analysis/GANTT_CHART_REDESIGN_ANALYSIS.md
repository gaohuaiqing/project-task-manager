# 甘特图时间计划重设计方案

> 创建时间：2025-03-04
> 状态：设计中

## 需求分析

### 用户期望的核心功能
1. **可视化拖拽编辑** - 能够拖拽任务条调整开始/结束日期
2. **时间线可视化** - 清晰显示项目时间轴、任务条、依赖关系
3. **缩放和导航** - 能够缩放时间轴（日/周/月视图），快速导航到特定日期
4. **双击快速编辑** - 双击里程碑/任务快速编辑详细信息

### 交互体验设计

#### 拖拽体验：实时跟随模式
- 拖拽时任务条跟随鼠标移动
- 显示实时位置和日期信息
- 松开鼠标后应用更改
- ESC键取消拖拽

#### 时间线布局：纯时间轴视图
```
┌─────────────────────────────────────────────────────────────┐
│  2025年1月                    2025年2月                      │
│  ─┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬─  │

│  🔵 前端开发 ─────────────────────────  [赵六]                │
└─────────────────────────────────────────────────────────────┘
```

#### 任务层级：扁平列表显示
- 所有任务平铺显示
- 不显示父子层级关系
- 使用颜色或标签区分任务类型

#### 里程碑交互：统一时间轴
- 里程碑和任务在同一时间轴
- 里程碑用菱形/圆形图标表示
- 任务用矩形条表示

#### 缩放粒度：无级缩放
- 滑块连续缩放
- 像素/天比例：5px ~ 100px
- 实时更新刻度标签

#### 视觉风格：现代简约风格
- 配色方案参考 Notion/Linear
- 柔和的背景色和边框
- 充足的留白和间距
- 清晰的层次结构

## 技术架构

### 组件结构
```
ModernGanttView/              # 新的甘特图容器
├── ModernGanttHeader/         # 顶部工具栏（缩放、导航）
├── ModernGanttTimeline/       # 时间轴刻度层
├── ModernGanttCanvas/         # 画布层（任务条、里程碑）
├── ModernGanttTooltip/        # 悬浮提示
└── ModernGanttEditor/         # 双击编辑器
```

### 核心Hook
```
useGanttCanvas/                # 画布状态管理
useGanttDrag/                  # 拖拽状态管理
useGanttZoom/                  # 缩放状态管理
useGanttNavigation/            # 导航状态管理
```

### 工具函数
```
ganttGeometry.ts               # 几何计算
ganttDragging.ts               # 拖拽计算
ganttRendering.ts              # 渲染辅助
```
