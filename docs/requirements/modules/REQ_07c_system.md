## 1. 系统配置

### 1.1 配置模块概述

**来源**: FINAL_REQUIREMENTS_0404-0827#L2002-2036

**优先级**: P1
**实现状态**: ⚠️ 部分实现（有环境变量，需要开发配置界面）

**功能概述**: 提供系统配置管理界面，支持项目类型、任务类型、节假日、组织架构等配置项的运行时调整。

**第二层 - 内容区顶部Tab导航**:
```

```

## 2. 导入导出

### 2.1 导入导出模块概述

**来源**: FINAL_REQUIREMENTS_0404-0827#L2057-2111, UI_Requirement_0404-0827#L3432-3593

**优先级**: P2
**实现状态**: ⏳ 待开发
**UI决策状态**: ✅ 8/8 已确认

**功能概述**: 提供4个领域的数据导入导出功能，支持Excel/CSV/JSON格式。

---

### 2.2 UI决策汇总表

| 决策点 | 最终选择 | 说明 |
|--------|----------|------|
| 文件上传方式 | **拖拽 + 点击双支持** | 灵活性强，用户体验最佳 |
| 导入进度显示 | **页面内进度条** | 不遮挡内容，可同时操作其他功能 |
| 错误处理展示 | **对话框内列表** | 立即反馈，便于修正后重试 |
| 导出选项布局 | **对话框** | 符合用户习惯，快速设置 |
| 字段选择方式 | **拖拽排序** | 可调整列顺序，灵活性强 |
| 导出格式选择 | **单选按钮** | 直观明确，快速选择 |
| 进度对话框 | **可取消** | 用户可控，防止误操作 |
| 四领域入口 | **工具栏下拉菜单** | 节省空间，入口统一 |

---

### 2.3 四个导入导出领域

| 领域 | 导出功能 | 导入功能 | 入口位置 |
|------|----------|----------|----------|
| 🏢 项目管理领域 | ✅ | ✅ | 项目列表页工具栏 |
| 📋 任务管理领域 | ✅ | ✅ | 任务管理页工具栏 |
| 👥 组织与权限领域 | ✅ | ✅ | 成员管理页工具栏 |
| ⚙️ 系统管理领域 | ✅ | ✅ | 各配置页工具栏 |

**导出内容明细**:
- 🏢 项目管理：项目、里程碑、项目成员
- 📋 任务管理：WBS任务、任务依赖、进度记录

- ⚙️ 系统管理：配置项、节假日、技能字典

---

### 2.4 导入功能

#### 2.4.1 文件选择区域

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              📁 拖拽文件到此处，或                    │
│                                                     │
│              [ 点击选择文件 ]                        │
│                                                     │
│         支持 .xlsx, .csv, .json 格式                │
└─────────────────────────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 触发方式 | 拖拽 + 点击按钮双支持 |
| 文件类型限制 | .xlsx, .csv, .json |
| 视觉样式 | 虚线边框 + 浅蓝背景 + 拖拽高亮 |
| 文件大小限制 | 最大10MB |

#### 2.4.2 导入进度显示（页面内）

```
┌─────────────────────────────────────────────────────┐
│ 导入任务数据                              65%       │
│ ████████████████████░░░░░░░░░░░░░░░░░░░             │
│ 正在导入... 13/20                                   │
│ ✅ 成功: 12  ❌ 失败: 1                              │
└─────────────────────────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 位置 | 页面内嵌进度条 |
| 进度条 | Progress组件，显示百分比 |
| 状态文字 | "正在导入... (X/Y行)" |
| 统计信息 | 实时显示成功/失败数量 |
| 取消按钮 | 可取消导入 |

#### 2.4.3 导入错误处理

```
┌─────────────────────────────────────────────────────┐
│ ❌ 导入错误                              [3个错误]  │
├─────────────────────────────────────────────────────┤
│ 行3: 项目编码重复                                   │
│ 行5: 日期格式错误                                   │
│ 行8: 成员不存在                                     │
├─────────────────────────────────────────────────────┤
│ [导出错误报告]                    [重新导入] [继续] │
└─────────────────────────────────────────────────────┘
```

| 错误类型 | 显示方式 |
|----------|----------|
| 格式错误 | 对话框内列表显示 |
| 数据验证错误 | 显示行号 + 错误原因 |
| 用户选择 | 可选择继续导入或取消 |

#### 2.4.4 导入核心流程

```
导入流程:
1. 选择文件 → 解析 → 校验 → 发现错误? → 暂停提示用户
                                    ↓
                              用户选择: 继续/取消
                                    ↓
2. 智能合并 → 写入数据库 → 完成提示
```

### 2.8 详细需求确认

| 决策项 | 最终选择 | 说明 |
|--------|----------|------|
| 使用场景 | 全部支持 | 数据迁移、批量维护、数据备份 |
| 文件格式 | Excel + CSV + JSON | 兼顾兼容性和系统集成 |
| 冲突策略 | 智能合并 | 自动合并新旧数据，冲突时提示用户 |
| 数据校验 | 用户选择模式 | 发现错误时暂停，让用户决定继续/取消 |
| 字段偏好 | 不保存 | 每次导出重新选择字段 |
| 下载模板 | 示例模板 | 提供带示例数据的模板，方便理解字段含义 |
| 操作日志 | 不记录 | 简单实现，导入导出完就结束 |
| 权限控制 | 按领域控制 | 各领域的导入导出权限与该领域的数据权限一致 |
