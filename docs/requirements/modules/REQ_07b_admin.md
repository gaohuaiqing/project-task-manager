# 模块需求：报表分析 - admin角色视图 (07b-admin)

> **模块编号**: 07b-admin
> **模块名称**: 报表分析 - admin（系统管理员）角色视图
> **父模块**: [REQ_07b_reports.md](./REQ_07b_reports.md)

---

## 1. 角色定位

**角色**: admin（系统管理员）
**核心视角**: 全局汇总 + 部门对比 + 资源调配

**数据范围**:
- 全部项目
- 全部成员
- 全部部门

**默认报表**: 项目进度报表
**默认筛选**: 全部项目

---

## 2. 报表视图定义

### 2.1 项目进度报表 - admin视图

**默认筛选**：
- 项目：全部项目
- 时间范围：过去30天
- 项目类型：全部

**差异化内容**：
| 组件 | 内容 |
|------|------|
| 统计卡片 | 全局项目总数、全局任务总数、全局完成率、全局延期率 |
| 图表1 | 全部项目任务状态分布 |
| 图表2 | 项目类型进度对比（产品开发/职能管理/物料改代/质量处理） |
| 动态图表 | 全局进度趋势、各部门进度变化速度对比 |
| 数据表格 | 全部项目里程碑列表，支持按部门筛选 |

---

### 2.2 任务统计报表 - admin视图

**默认筛选**：
- 项目：全部项目
- 负责人：全部
- 任务类型：全部

**差异化内容**：
| 组件 | 内容 |
|------|------|
| 统计卡片 | 全局任务总数、全局平均完成率、全局延期率、全局紧急任务数 |
| 图表1 | 任务类型分布（12类）全局统计 |
| 图表2 | 部门任务分布对比柱状图 |
| 动态图表 | 全局任务新增/完成/延期趋势，按部门分解 |
| 数据表格 | 全部成员任务统计，支持按部门筛选 |

---

### 2.3 延期分析报表 - admin视图

**默认筛选**：
- 项目：全部项目
- 时间范围：过去30天
- 延期类型：全部

**差异化内容**：
| 组件 | 内容 |
|------|------|
| 统计卡片 | 全局延期任务总数、全局延期预警数、全局已延迟数、全局超期完成数 |
| 图表1 | 部门延期分布对比柱状图 |
| 图表2 | 延期原因分类统计（全局） |
| 动态图表 | 各部门延期趋势对比、新增延期vs已解决趋势 |
| 数据表格 | 全部延期任务列表，支持按部门筛选 |
| 特有功能 | 跨部门资源调配建议（识别延期高发部门、闲置部门） |

---

### 2.4 成员任务分析 - admin视图

**默认筛选**：
- 成员：全部（对比视图）
- 时间范围：过去30天

**差异化内容**：
| 组件 | 内容 |
|------|------|
| 统计卡片 | 全局平均任务负载、全局平均全职比、全局平均完成率、全局能力匹配度 |
| 图表1 | 部门成员负载分布（堆叠柱状图） |
| 图表2 | 部门效能对比（散点图：X=负载，Y=完成率） |
| 动态图表 | 各部门负载变化趋势对比 |
| 数据表格 | 全部成员任务明细，支持按部门筛选 |

---

### 2.5 资源效能分析 - admin视图

**差异化内容**：
| 组件 | 内容 |
|------|------|
| 统计卡片 | 全局平均产能、全局平均预估准确性、全局平均返工率、全局全职比利用率 |
| 图表1 | 部门产能排名对比 |
| 图表2 | 部门效能四象限（产能 × 准确性） |
| 数据表格 | 全部成员效能明细，支持按部门筛选 |
| 特有功能 | 资源调配建议（识别瓶颈部门、闲置部门） |
