### 5.4 导入功能

#### 5.4.1 文件选择区域

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              📁 拖拽文件到此处，或                    │
│                                                     │
│              [ 点击选择文件 ]                        │
│                                                     │
│         支持 .xlsx, .csv, .json 格式                │
└─────────────────────────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 触发方式 | 拖拽 + 点击按钮双支持 |
| 文件类型限制 | .xlsx, .csv, .json |
| 视觉样式 | 虚线边框 + 浅蓝背景 + 拖拽高亮 |
| 文件大小限制 | 最大10MB |

#### 5.4.2 导入进度显示（页面内）

```
┌─────────────────────────────────────────────────────┐
│ 导入任务数据                              65%       │
│ ████████████████████░░░░░░░░░░░░░░░░░░░             │
│ 正在导入... 13/20                                   │
│ ✅ 成功: 12  ❌ 失败: 1                              │
└─────────────────────────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 位置 | 页面内嵌进度条 |
| 进度条 | Progress组件，显示百分比 |
| 状态文字 | "正在导入... (X/Y行)" |
| 统计信息 | 实时显示成功/失败数量 |
| 取消按钮 | 可取消导入 |

#### 5.4.3 导入错误处理

```
┌─────────────────────────────────────────────────────┐
│ ❌ 导入错误                              [3个错误]  │
├─────────────────────────────────────────────────────┤
│ 行3: 项目编码重复                                   │
│ 行5: 日期格式错误                                   │
│ 行8: 成员不存在                                     │
├─────────────────────────────────────────────────────┤
│ [导出错误报告]                    [重新导入] [继续] │
└─────────────────────────────────────────────────────┘
```

| 错误类型 | 显示方式 |
|----------|----------|
| 格式错误 | 对话框内列表显示 |
| 数据验证错误 | 显示行号 + 错误原因 |
| 用户选择 | 可选择继续导入或取消 |

#### 5.4.4 导入核心流程

```
导入流程:
1. 选择文件 → 解析 → 校验 → 发现错误? → 暂停提示用户
                                    ↓
                              用户选择: 继续/取消
                                    ↓
2. 智能合并 → 写入数据库 → 完成提示
```
